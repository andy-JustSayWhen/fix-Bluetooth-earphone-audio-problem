# fix-Bluetooth-earphone-audio-problem

本项目用户排查并解决macOS和Windows连接带麦克风的蓝牙耳机时，播放音频期间出现的音质异常问题。所谓“异常”听感为：有些时候，声音听起来很空洞、失真、码率很低、传输不稳定或者压根没声音等问题。你确信耳机没坏，系统设置没乱改，而且你反复在设置内切换输出和输入设备，有时能解决有时又完全没用。

## 项目目录

项目目录结构和新增、移动文件的规则统一维护在 [`reference/项目目录树.md`](reference/项目目录树.md)。其中 `test/` 的一级目录按完整测试案例划分；同一案例的原始文件和测试产物放在同一个案例目录内，不按文件类型建立一级目录。
